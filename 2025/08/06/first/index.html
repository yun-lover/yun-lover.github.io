项目简介

LingChat-C++-Termux 是开源项目 LingChat 的一个 C++ 移植版本，专为 Termux 环境打造。它集成了 RAG (检索增强生成) 系统，为 AI 对话提供长期记忆能力，并可以高度自定义。目前主要在 Termux 中测试运行，其他 Linux 环境需自行调整编译配置。

注意事项

  1. 依赖问题： 确保 Termux 安装了所有必需的库。
  2. API 配置： .env 文件中的 API 配置是核心，务必正确填写。
  3. RAG 系统： 如需启用 RAG (ENABLE_RAG = true)，需额外配置 EMBEDDING_MODEL, EMBEDDING_API_URL 和 EMBEDDING_VECTOR_DIMENSION。
  4. 资源路径： 图片、音效路径可在 .env 的 [UI] 部分自定义。
  5. 情绪映射： [EmotionMap] 和 [EmotionSfxMap] 定义了表情与资源文件的对应关系。
  6. 语音合成： 相关功能 (VOICE_API_URL) 因性能问题已废弃。
  7. 调试信息： 终端输出的 CivetWeb 日志 (如 *** 1753620119.065422618 ...) 通常可忽略。
  8. 预编译版本： 仓库中的 backend_server 是在下编译版本，如运行报错请自行编译。
